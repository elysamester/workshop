Sobre o Workshop

Este workshop é projetado para proporcionar um conhecimento profundo do HTML e CSS, usando técnicas modernas e práticas recomendadas. Ideal para quem deseja avançar na carreira de desenvolvimento web.

Agenda

  • Dia 1: Introdução ao HTML5 e CSS3
  • Dia 2: Layouts Dinâmicos com Flexbox
  • Dia 3: Design Avançado com CSS Grid
  • Dia 4: Projeto Final e Avaliação
